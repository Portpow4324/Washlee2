### For Backend API
```json
{
  "delivery": {
    "addressLine1": "123 Main Street",
    "addressLine2": "Apt 10",
    "city": "Sydney",
    "state": "NSW",
    "postcode": "2000",
    "country": "Australia"
  }
}
```

```json
{
  "address": "123 Main Street, Apt 10",
  "suburb": "Sydney",

}
```
